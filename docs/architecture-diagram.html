LLMHub Architecture

Electron + TypeScript + React + SQLite Desktop Application

Frontend (Renderer)

  • • React 18 + TypeScript
  • • Vite for fast HMR builds
  • • Tailwind CSS + macOS styling
  • • Zustand for state management
  • • Tab-based UI navigation

Backend (Main Process)

  • • Electron 28+ main process
  • • IPC handlers for secure data access
  • • better-sqlite3 for local storage
  • • AES-256-GCM key encryption
  • • Profile & provider management

Database Schema

  • • profiles: Multi-workspace support
  • • providers: LLM API configurations
  • • models: Names, pricing, token limits
  • • api_keys: Encrypted credentials
  • • Stored in ~/Library/App Support/

Supported Providers

  • • OpenAI (GPT-4, GPT-3.5)
  • • Anthropic (Claude 3, 3.5)
  • • Google AI (Gemini)
  • • Azure OpenAI Service
  • • Local: Ollama, LM Studio, vLLM

Code Generation

  • • Python (openai, anthropic SDKs)
  • • curl commands
  • • JavaScript/TypeScript
  • • Chat, streaming, tool examples
  • • Copy-to-clipboard ready

Security Features

  • • API keys encrypted at rest
  • • Electron safeStorage integration
  • • Context isolation enabled
  • • No remote code execution
  • • Local-only data storage
